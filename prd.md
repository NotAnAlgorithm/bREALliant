
# bREALliant analysis

Brilliant-like learning app for studying real analysis.

- need a focused, smooth, clean UI to facilitate the learning
- abstracted architecture and custom format for designing lessons (JSON or similar?)
  - easily and quickly iterate and develop courses without needing to dig into code
  - modular setup for pages, responsive simulations/activities/sliders/buttons, quizzes, etc
  - easy to design archetypes of interactive diagrams
- simple but compelling features to motivate students

comprehensive testing suite
- unit tests for everything
- both visual and non-visual verifiability, maybe human-in-the-loop
- exhaustive performance tests too
- everything a user may do to tinker with it should work

## user persona

### undergraduate trying to learn real analysis

- math or math-adjacent major
- 16-20 years old
- interested in math and decently proficient, fast learner

### graduate (student) learning real analysis for something else

- not a math major
- 21-25 years old
- wanting to learn analysis in service of another purpose, not necessarily out of interest alone
- struggles a bit more with complex mathematics
